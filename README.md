# Wechat_bot

## 产品期待

辅助微信社群运营。

请**合法合规**使用，本代码及作者不对任何事件后果负责，相关AI生成后的结果均为模型生成，不代表作者观点和看法。

1. 支持日常微信管理操作，包括不限于：自动添加好友；自动拉好友入群；群@；群发消息等；

2. 支持部分特定问题的回复，如平台使用问题、日常对话；

3. 支持部分文娱，如AIGC等；

4. 待续。
